# WebProject : 도그위즈
### 애견인들의 커뮤니티를 형성 서로 산책경로를 공유하고 커뮤니티를 이용한 웹 구현

### ◆ 개발 환경
* Java 8
* JDK 1.8.0
* Server : Apache Tomcat 9.0
* Database : MySQL

### ◆ 학습 목표
* JavaScript 웹 개발 능력치 향상
* 데이터베이스 및 서버 관리 향상
* Public Web API 사용

### ◆ 담당 업무
* 연령대 설정(MZ)
* 홈페이지 이름 / 디자인 설계(총괄)
* Public Web API ( 지도, 위치, 경로, 날씨 )
* 데이터베이스 정규화 및 모델링
* 게시판 목록 구현 : 자바스크립트를 이용해서 페이지 구현
